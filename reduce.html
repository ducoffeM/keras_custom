Keras Custom Reduce

class keras_custom.layers.reduce.base_reduce.BaseAxisKeepdimsLayer(*args, **kwargs)

Base Keras layer providing axis and keepdims attributes for layers performing reduction operations. This layer serves as an abstract base class, allowing subclasses to inherit and use these attributes.

class keras_custom.layers.reduce.max.Max(*args, **kwargs)

Custom Keras Layer that computes the maximum value along a specified axis of the input tensor. Inherits axis and keepdims attributes from BaseAxisKeepdimsLayer.

class keras_custom.layers.reduce.max.Argmax(*args, **kwargs)

Custom Keras Layer that computes the index of the maximum value along a specified axis. Inherits axis and keepdims attributes from BaseAxisKeepdimsLayer.

class keras_custom.layers.reduce.min.Min(*args, **kwargs)

Custom Keras Layer that computes the minimum value along a specified axis. Inherits axis and keepdims attributes from BaseAxisKeepdimsLayer.

class keras_custom.layers.reduce.mean.Mean(*args, **kwargs)

Custom Keras Layer that computes the mean of elements along a specified axis. Inherits axis and keepdims attributes from BaseAxisKeepdimsLayer.

class keras_custom.layers.reduce.sum.Sum(*args, **kwargs)

Custom Keras Layer that computes the sum of elements along a specified axis. Inherits axis and keepdims attributes from BaseAxisKeepdimsLayer.
