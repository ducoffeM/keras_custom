Keras Custom Activations

class keras_custom.layers.activations.activation.Clip(*args, **kwargs)

Custom Keras Layer that clips the values of a Keras Tensor element-wise, within a specified range.

This layer performs element-wise clipping, setting values below a given minimum (vmin) to that minimum and values above a given maximum (vmax) to that maximum. This is useful for constraining values in a neural network layer to a fixed range during forward propagation.

Example:

import tensorflow as tf

clip_layer = Clip(vmin=0.0, vmax=1.0)
input_data = tf.constant([[-1.0, 0.5, 2.0]])
output_data = clip_layer(input_data)
# Output will be: [[0.0, 0.5, 1.0]]
class keras_custom.layers.activations.activation.Floor(*args, **kwargs)

Custom Keras Layer that applies the floor operation to each element in the input tensor.

This layer rounds down each element in the input tensor to the nearest integer. It is often used in scenarios where discrete, integer values are needed, or where fractional components of tensor elements should be removed.

Example:
import tensorflow as tf

floor_layer = Floor()
input_data = tf.constant([[1.7, 2.9, -0.3]])
output_data = floor_layer(input_data)
# Output will be: [[1.0, 2.0, -1.0]]

class keras_custom.layers.activations.activation.Log(*args, **kwargs)

Custom Keras Layer that applies the natural logarithm (log) operation to each element in the input tensor.

This layer performs an element-wise natural logarithm on the input tensor, which is useful for various mathematical and preprocessing applications where log transformations are applied to data. Note that input values must be strictly positive, as the log of non-positive numbers is undefined.

Example:
import tensorflow as tf

log_layer = Log()
input_data = tf.constant([[1.0, 2.0, 10.0]])
output_data = log_layer(input_data)
# Output will be approximately: [[0.0, 0.6931, 2.3026]]
